# GymLabz Hub

O **GymLabz Hub** é um sistema completo e moderno focado na gestão inteligente de academias, estúdios de crossfit e centros de treinamento. Desenvolvido para centralizar e facilitar todas as operações diárias, ele oferece um painel administrativo intuitivo tanto para gestores quanto para instrutores e recepcionistas.

## 🚀 Funcionalidades Principais

- **Gestão de Alunos**: Cadastro, histórico, perfis e acompanhamento de evolução.
- **Treinos Personalizados**: Criação, edição e atribuição de fichas de treino sob medida para cada aluno.
- **Gestão Financeira e Assinaturas**: Controle inteligente de planos, mensalidades, inadimplentes, além de gráficos de faturamento mensal.
- **Controle de Acesso**: Gerenciamento integrado de entradas, saídas e regras de catraca para alunos com pagamentos pendentes.
- **Grade de Aulas e Matrículas**: Organização de modalidades, horários de pico e vagas disponíveis.
- **Gestão de Funcionários**: Cadastro de equipe (professores, gerentes, recepcionistas) com controle de papéis e permissões (RBAC).
- **Comunicação Interna**: Mural de avisos diretos para o aplicativo dos alunos.

## 💻 Painel Administrativo

A interface web foi desenhada com fortes princípios de usabilidade, proporcionando acesso rápido a:
- Relatórios Financeiros (Receita Certa, Pendente, Inadimplente)
- Métricas de Ocupação e Capacidade da Academia
- Edição dos Dados Cadastrais da Empresa

## 🛠️ Tecnologias Utilizadas

Este projeto front-end foi construído utilizando as seguintes tecnologias:
- **[React](https://reactjs.org/)** com **TypeScript**
- **[Vite](https://vitejs.dev/)** para tooling e build rápido
- **[Tailwind CSS](https://tailwindcss.com/)** para estilização utilitária e design responsivo (com suporte a Dark Mode)
- **[React Router DOM](https://reactrouter.com/)** para navegação aninhada e rotas protegidas
- **[Lucide React](https://lucide.dev/)** para ícones modernos

## 📦 Como Executar o Projeto

1. Clone o repositório.
2. Acesse a pasta do projeto: `cd gymlabz-hub`
3. Instale as dependências executando:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento local:
   ```bash
   npm run dev
   ```
5. Acesse `http://localhost:5173` no seu navegador.
